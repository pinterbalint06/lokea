1. lépés

Egyedek:
- users
- images
- game_maps
- map
- points
- scores

2. lépés
Kapcsolatok:

- upload_map (users, game_maps): 1-N kapcsolat, mert egy felhasználó több game mapot is létrehozhat, viszont egy game map egy készítőhöz tartozik.

- contains (game_maps, map): 1-N kapcsolat, mert egy game_map több mapból állhat, de egy térképrészlet egy game_maphoz tartozik.

- image_of_map (map, images): 1-N kapcsolat, mert egy térképhez több kép is tartozhat, de egy kép csak egy térképhez tartozik.

- use_map (map, points): 1-N kapcsolat, mert egy térképen több pont is elhelyezkedhet, viszont egy pont csak egy térképhez tartozik.

- use_pictures (points, images): 1-1 kapcsolat, mert egy ponthoz pontosan egy kép tartozik.

- score_points (users, scores): 1-N kapcsolat, mert egy felhasználónak több pontszáma is születhet, viszont egy pontszám rekord egy felhasználóhoz kötődik.

- score_calculation (game_maps, scores): 1-N kapcsolat, mert egy game_maphoz több pontszám is tartozhat, de egy pontszám egy adott game_maphoz kapcsolódik.

3. lépés
Egyed táblák:
- users: _userid_, username, email, password, role, pfp, is_2fa, created_at, deleted_at
- images: _image_id_, width, height, filepath
- game_maps: _game_maps_id_, title, game_created
- map: _map_id_
- points: _point_id_, point_x, point_y
- scores: _score_id_, score, score_time

Kapcsolat táblák:
- upload_map: _game_maps_id_, userid
- contains: _map_id_, game_maps_id
- image_of_map: _map_id_, image_id
- use_map: _point_id_, map_id
- use_pictures: _point_id_, image_id
- score_points: _score_id_, userid
- score_calculation: _score_id_, game_maps_id

4. lépés
Táblák összevonása

Az 1:N (egy a többhöz) kapcsolatok esetén a kapcsolatot nem külön táblaként valósítjuk meg,
hanem beolvasztjuk a „több” (N) oldalon álló egyed táblájába idegen kulcsként (Foreign Key).
Mivel a rendszerben minden kapcsolat 1:N típusú (vagy 1:1, ami szintén összevonható),
a 3. lépésben felírt kapcsolótáblák mindegyike beépül a fő táblákba.

- users: _userid_, username, email, password, role, pfp, is_2fa, created_at, deleted_at
- images: _image_id_, width, height, filepath
- game_maps: _game_maps_id_, creator_id, title, game_created
- map: _map_id_, game_maps_id, image_id
- points: _point_id_, map_id, point_x, point_y, image_id
- scores: _score_id_, user_id, game_maps_id, score, score_time
